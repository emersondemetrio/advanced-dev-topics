Advanced DEV Topics

## Topics

- **Debounce** - Function that delays execution until a specified time has passed since last invocation
- **Deep Copy** - Implementation of deep copying objects/arrays
- **Events** - Event emitter pattern implementation with custom EventEmitter class
- **HTTP-FS** - Comparing HTTP requests vs file system operations (Python & TypeScript)
- **Migrations** - Database migration system example with schema versioning
- **Promise All & Promise All Settle** - Custom implementations of Promise.all() and Promise.allSettled()
- **Sorting (Bubble Sort)** - Classic bubble sort algorithm implementation
- **Threads** - Multi-threading examples and concurrent execution patterns (Python & TypeScript)
